# WhatsApp Reminder Bot 🚀

## Setup

1. Install Node.js
2. Clone repo
3. Run:
   npm install
4. Run:
   node index.js
5. Scan QR from WhatsApp

## Features
- Send automated WhatsApp messages
- Works for groups & personal chats
- Schedule reminders

## Deployment
Deploy easily on Railway for 24/7 running.
